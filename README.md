# 🛒 Retail Sales Forecasting & Inventory Optimization System

## 📌 Overview
This project simulates a real-world retail analytics system that transforms raw sales data into actionable insights. It predicts future demand using machine learning and optimizes inventory using safety stock and reorder point logic.

---

## 🚀 Features
- Data generation and preprocessing pipeline  
- Exploratory Data Analysis (8 professional charts)  
- Sales forecasting using Machine Learning  
- Inventory optimization (Safety Stock & Reorder Point)  
- Interactive filtering and insights  

---

## 🛠️ Tech Stack
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-learn  
- Streamlit  

---

## 📊 Project Workflow
1. Data Generation  
2. Data Cleaning & Preprocessing  
3. Exploratory Data Analysis  
4. Feature Engineering  
5. Sales Forecasting  
6. Inventory Optimization  
7. Visualization & Insights
